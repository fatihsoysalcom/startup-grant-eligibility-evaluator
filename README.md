# Startup Grant Eligibility Evaluator

This example demonstrates a simplified grant evaluation system for startups. It defines various criteria (like social impact, innovation, and market potential) with assigned weights, simulating how grant-giving organizations prioritize different aspects of a startup project. The program processes hypothetical startup applications, calculates a weighted score for each, and determines eligibility based on a predefined threshold.

## Language

`python`

## How to Run

Save the code as `grant_evaluator.py`. Open a terminal or command prompt, navigate to the directory where you saved the file, and run `python grant_evaluator.py`.
